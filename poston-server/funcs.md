1. Админ
- Создаёт пользователей с ролями курьер,склад и постамат
- Создаёт каталог товаров
- Будет видеть статистику по пользованию приложения и доставкой

2. Покупатель
- Регистрация
- Может видеть каталог товаров
- Может добавить товары в корзину, просмотреть и удалить
- Возможность оформления заказа и оплаты, выбор типа доставки 
- Возможность видеть статус доставки оплаченного заказа
- Возможность забрать товар (при доставке курьером продиктовать код, при доставке постоматом отсканировать qr)

3. Курьер
- Выбор случайного самого свободного курьера и таким образом назначается заказ случайный склад на котором есть все товары в этом заказе
- Курьер получает уведомление, что ему назначен заказ (получение уведомления через telegram и в самом приложении)
- Может получить товар со склада для доставки
- Должен отвезти товар в течение определенного времени (если товар не будет доставлен в течение опред.времени - отправить предупреждение в приложении/telegram)
Пред (в последнюю очередь) - сделать отдельную таблицу, где будет id пользователя, на сколько курьер опоздал и с каким заказом это было связано (id заказа)
- Возможность просканировать qr в постамате и после этого в приложении будет написано, куда положить товары (в какие ячейки)
- Если доставка проводится лично - курьер спрашивает код и в приложении может ввести код данного пользователя
- Статистика доставок

4. Склад
- Возможность принимать товар на склад (создавать товар)
- Возможность выдавать товар курьеру, товар должен быть связан с id заказа, а он уже с курьером.
- Человек может забрать доставку напрямую со склада (ПОД ВОПРОСОМ)
- Просмотр наличия заказа

5. Постамат
- Отображает свой QR-код, который могут отсканировать покупатель/курьер и таким образом выдавать товар пользователю либо принимать товар от курьера.
- Звуковое сообщение "Пожалуйста закройте ячейку". (Реализация будет таковой : 3 секунды открыть, 3 секунды закрыть дверь, будут кнопки для этого)
